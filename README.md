# ❤️ 3D Сердце

Красивая 3D-анимация сердца из светящихся частиц на звёздном фоне.

## 🌟 Демо

Открыть сайт: [https://yuliapositive.github.io/love-heart/](https://yuliapositive.github.io/love-heart/)

## ✨ Особенности

- 💖 3D-сердце из частиц
- 💓 Пульсация (бьётся)
- 🔄 Вращение в 3D-пространстве
- ⭐ Мерцающие звёзды на фоне
- 🌈 Красный/розовый градиент

## 🚀 Как запустить локально

1. Скачайте файл `index.html`
2. Откройте его в браузере двойным кликом

## 🛠️ Технологии

- Three.js — 3D-графика
- JavaScript (ES Modules)
- GitHub Pages — хостинг
